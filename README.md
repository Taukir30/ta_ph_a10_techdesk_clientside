# Project Name:
    TechDesk - Freelancing Marketplace

## Overview:
    TechDesk is a robust, full-stack freelancing platform designed to bridge the gap between clients and skilled professionals. Built with the MERN stack, it provides a seamless ecosystem for task management and digital earning.

# Live Site Link: https://a10-tech-desk-client-side.netlify.app
    
# Features:
    *Account registration with email and password
    *Login with google
    *Posting job with cover image
    *Editing and deleting posted job
    *Can accept task view accepted tasks
    *Can complete or cancel accepted task
    *Secured private routes
    *Light and Dark theme toggle
    *Responsive Design

## Dependencies :
    1. Tailwind CSS
    2. DaisyUI
    3. Axios
    4. Firebase
    5. React Icons
    6. React Router
    7. Recharts
    8. Sweetalert2
    9. Toastify

## How to Run TechDesk Locally
    1. Make sure Node.js (v16+) and Git are installed
    2. Clone the repository:
        git clone https://github.com/Taukir30/ta_ph_a10_techdesk_clientside
    3. Go to the project folder:
        cd ta_ph_a10_techdesk_clientside
    4. Install dependencies:
        npm install
    5. Create a .env file if required and add environment variables (see .env.example)
    6. Start the development server:
        npm run dev
    7. Open your browser and visit:
        http://localhost:5173
